# Shubham Verma - Portfolio Website

A modern, professional portfolio website built with React.js showcasing my experience as an SDE 1, specializing in Mobile App Development, Website Development, and AI/ML Chatbot Integration.

## Features

- **Responsive Design**: Works seamlessly on all devices
- **Modern UI/UX**: Clean and professional design
- **Smooth Animations**: Engaging user experience
- **Sections**:
  - Hero section with introduction
  - About section with experience highlights
  - Professional experience (PRAAPTI, QuadB)
  - Featured projects
  - Technical skills
  - Contact form

## Tech Stack

- React.js
- CSS3 (Custom styling)
- React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

Update the following files to customize the portfolio:

- `src/components/Hero.js` - Hero section content
- `src/components/About.js` - About section
- `src/components/Experience.js` - Experience details
- `src/components/Projects.js` - Project listings
- `src/components/Skills.js` - Skills and expertise
- `src/components/Contact.js` - Contact information
